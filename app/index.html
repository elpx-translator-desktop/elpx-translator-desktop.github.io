Versión web

ELPX Translator web

Traduce archivos .elpx desde el navegador.

Si no dispones de clave API o prefieres trabajar en local, usa la versión de escritorio.

Qué hace esta versión

  • Traduce archivos .elpx.
  • Genera un nuevo archivo traducido para descargar.
  • Necesita tu propia clave API para funcionar.

API y modelo

La clave API se guarda en este navegador para que no tengas que escribirla cada vez.

Privacidad:

  • El archivo se abre localmente en el navegador.
  • El texto se envía al servicio API que elijas para hacer la traducción.
  • No hay un servidor propio intermedio.

¿No tienes API?

  • La versión web necesita una clave API de un proveedor externo.
  • Si no dispones de ella, usa la versión de escritorio con procesamiento local.
  • Ir a la página principal para ver información y descargas.
